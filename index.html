Welcome to Loomio.org

Loomio is a free open source web app that helps groups facilitate discussions and make decisions together. This page is dedicated to providing general information about the project and instructions for contributing. If you're looking for current updates about the project, head over to the blog at http://blog.loom.io.

Contribute

There are lots of different ways you can contribute to Loomio. If you're a web developer, you can start by following the instructions below. If you'd like to help out in some other way, feel free to just send us an email at contact@loom.io. We're friendly and are excited to hear from anyone who wants to help out. =)

Get started contributing (web developers)

Loomio is a Ruby on Rails application. If you're new to Ruby on Rails, you can have a read through our Loomio Getting Started document and it should help you get set up. If you're already familiar with Rails then the setup should be pretty simple. Just run the following commands and you should be good to go.

$ git clone git@github.com:enspiral/loomio.git
$ cd loomio
$ bundle install
$ cp config/database.example.yml config/database.yml
$ bundle exec rake db:setup; bundle exec rake db:setup RAILS_ENV=test;
$ bundle exec rake db:seed; bundle exec rake db:seed RAILS_ENV=test;

We keep track of project management using the Loomio Trello Board. Feel free to have a look at the board while you're checking out the code to see if there might be any tasks that you'd feel comfortable taking on.
